===============================================
         PACA 대화형 테스트 사용법
===============================================

🎯 최종 정리된 버전 - 필요한 파일만!

📁 남은 파일:
├── PACA_SIMPLE.bat     ← 🔥 이것만 실행하세요! (관리자 권한 불필요)
├── PACA_WORKS.bat      ← 관리자 권한 버전
└── paca_ascii_only.py  ← 메인 프로그램

===============================================
         🚀 사용법 (초간단)
===============================================

1️⃣ "PACA_SIMPLE.bat" 더블클릭
2️⃣ 대화 시작! (관리자 권한 불필요)

===============================================
         💬 테스트 예제
===============================================

✅ "python study"        → Python 학습
✅ "learn javascript"    → JavaScript 학습
✅ "react help"          → React 도움
✅ "machine learning"    → ML 학습

📋 명령어:
  📊 "stats"  → 통계 보기
  ❓ "help"   → 도움말
  🚪 "quit"   → 종료

===============================================
         ✨ 확인되는 기능
===============================================

🎯 학습 요청 자동 감지
   → "study", "learn" 키워드 인식

🏷️ 주제별 맞춤 응답
   → Python, JavaScript, React 등 구분

🧠 메타인지 점수 계산
   → 60-99점 추론 품질 표시

📊 실시간 통계 추적
   → 대화수, 학습요청수, 성공률

===============================================
         🔧 문제 해결 완료!
===============================================

✅ UTF-8 인코딩 문제 해결
✅ 입력 처리 문제 해결
✅ 프로그램 종료 문제 해결
✅ 관리자 권한 무한루프 해결
✅ 창이 뜨고 바로 꺼지는 문제 해결
✅ 불필요한 파일 모두 정리

🆘 만약 여전히 문제가 있다면:

❓ 창이 뜨고 바로 꺼지는 경우:
   → "PACA_SIMPLE.bat" 사용 (관리자 권한 불필요)
   → Python이 제대로 설치되었는지 확인

❓ "Python not found" 오류:
   → https://python.org 에서 Python 설치
   → PATH에 Python이 등록되었는지 확인

❓ 파일을 찾을 수 없다는 오류:
   → 모든 파일이 같은 폴더에 있는지 확인
   → 폴더 경로에 한글이 있으면 영문으로 변경

===============================================
         🎉 이제 사용하세요!
===============================================

"PACA_WORKS.bat" 더블클릭 → 바로 시작!

문제 발생시:
- Python 설치 확인
- 관리자 권한으로 실행
- 파일이 같은 폴더에 있는지 확인
